# 개인정보처리방침 (RUNNERS)

RUNNERS(이하 “서비스”)는 「개인정보 보호법」 등 관련 법령을 준수하며, 이용자의 개인정보를 보호하기 위해 아래와 같이 개인정보처리방침을 수립·공개합니다.

- 시행일: 2026-01-21

## 1. 개인정보의 처리 목적

서비스는 다음의 목적을 위하여 개인정보를 처리합니다.

- 회원 식별 및 로그인/인증, 부정 이용 방지
- 서비스 제공(프로필, 커뮤니티 기능 등) 및 이용자 문의 대응
- 서비스 품질 개선 및 장애 대응(접속/오류 기록 등)
- (선택) 러닝·운동 기록 연동/조회 기능 제공(Health Connect 등)

## 2. 처리하는 개인정보 항목

서비스는 최소한의 개인정보만을 수집·이용합니다.

### 2.1 필수 수집 항목

| 구분 | 수집 항목 | 수집 방법 |
| --- | --- | --- |
| 회원/로그인(구글) | 이메일, Google 계정 고유 식별자(sub), 이름(프로필 표시명), 프로필 사진 URL | Google Sign-In을 통한 로그인 시 이용자가 제공/동의 |
| 서비스 이용 | 사용자 ID, 닉네임, 게시글/댓글 등 이용자가 작성·등록한 콘텐츠 | 회원가입/서비스 이용 과정에서 이용자가 직접 입력 |
| 보안/인증 | (서버) 리프레시 토큰, (앱) 액세스 토큰/리프레시 토큰(기기 내 저장) | 로그인/자동 로그인 과정에서 생성·저장 |

### 2.2 선택 수집 항목(이용자 동의 시)

| 구분 | 수집 항목 | 수집 방법 |
| --- | --- | --- |

### 2.3 자동 생성·수집 항목

서비스 이용 과정에서 아래 정보가 자동 생성·수집될 수 있습니다.

- 접속 로그(접속 일시, IP 주소, 사용자 에이전트/기기 정보), 오류 기록, 요청/응답 관련 기록
- 커뮤니티 이용 기록(예: 조회 여부 등 부정 이용 방지 및 기능 제공을 위한 최소 기록)

## 3. 개인정보의 처리 및 보유 기간

서비스는 개인정보 보유·이용기간이 경과하거나 처리 목적이 달성된 경우 지체 없이 해당 개인정보를 파기합니다.

| 구분 | 보유 기간 |
| --- | --- |
| 회원 정보(이메일, 닉네임 등) | 회원 탈퇴(계정 삭제) 시까지 |
| 커뮤니티 콘텐츠(게시글/댓글) | 원칙적으로 삭제 요청 또는 회원 탈퇴 시 처리(정책에 따라 익명화하여 보관할 수 있음) |
| 리프레시 토큰(서버 저장) | 토큰 만료 시 또는 로그아웃/탈퇴 시까지 |
| 접속/오류 기록 | 보안 및 서비스 안정화 목적 범위 내에서 필요한 기간 동안 보관 후 삭제 |

※ 법령에 따라 보존이 필요한 경우 해당 법령에서 정한 기간 동안 보관할 수 있습니다.
